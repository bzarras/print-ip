# print-ip
A command line tool for getting your IP on the LAN

## Install
```shell
$ npm install -g print-ip
```

## Usage
```shell
$ ip

172.104.0.3
```
or
```shell
$ print-ip

172.104.0.3
```

## Fun
You can append a color to the end of the command and it will print in that color
```shell
$ ip magenta

172.104.0.3 # will be the color magenta
```
Valid colors are:
* `black`
* `red`
* `green`
* `yellow`
* `blue`
* `magenta`
* `cyan`
* `white`
* `gray`
